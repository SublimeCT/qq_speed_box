# qq_speed_box

**⚠️ 本项目当前处于早期开发阶段, 待功能完善后再公布细节**

## 简介
- 这是一个 **QQ飞车(端游)** 赛王记录的数据库, 主要用于数据(视频)展示
- 所有数据都来自 [QQ飞车官方论坛视频区](https://speed.gamebbs.qq.com/forum.php?mod=forumdisplay&fid=30673) 的置顶记录帖
- 基于 `flutter` 开发
